# Spending Tracker

A mobile app for tracking personal expenses, built with React Native and Expo.

## Development

### Prerequisites
- Node.js 18+
- npm
- Expo CLI

### Setup
```bash
# Install dependencies
npm install

# Start development server
npm start

# Run on web
npm run web

# Run on iOS
npm run ios

# Run on Android
npm run android
```

### Environment Variables
Copy `.env.example` to `.env` and fill in the required values:
```bash
cp .env.example .env
```

### Testing
```bash
# Run linter
npm run lint

# Format code
npm run format
```
